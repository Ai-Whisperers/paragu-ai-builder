# Nexa Paraguay — static assets

This folder holds logo, photography and team photos. Keep images optimised:

| Asset | Target | Size cap |
|---|---|---|
| `logo.svg` + `logo-mono.svg` | Final Nexa Paraguay brand | <30 KB each |
| `hero-asuncion.jpg` | Hero / homepage background, 1920×1080 | <200 KB (webp) |
| `process-*.jpg` | Supporting process imagery, 1200×800 | <150 KB (webp) |
| `team-*.jpg` | Team member headshots, 600×600 | <80 KB (webp) |

Until commissioned photography arrives, use premium stock with a clearance
note in the PR description. Avoid: generic stock of office handshakes,
low-res Paraguay postcards, agency-of-travel aesthetic, government iconography.

Photography brief: see `relocation-business/source-of-truth/cuestionario-marketing-imagen-web.md`
questions 5 and 21 for the full visual brief (navy/champagne mood, modern
Asunción, editorial, lots of white space).
